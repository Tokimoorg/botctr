# Clan Rankings CTR - Bot & Web

Project for managing clan rankings and a Discord ticket system.

## Components

- **Discord Bot**: `index.js` handles ticket management and ranking updates.
- **Website**: A premium web interface for displaying clan rankings (`index.html`, `style.css`, `script.js`).

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure environment variables:
   Create a `.env` file with your bot token and other sensitive data.
3. Run the bot:
   ```bash
   node index.js
   ```
4. Open `index.html` in your browser to view the website.
